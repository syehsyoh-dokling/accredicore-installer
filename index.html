Native workflow

  1. Run requirement check.
  2. Install missing dependencies.
  3. Resolve any port conflicts.
  4. Clone accredicore-main into the target working directory.
  5. Create the database automatically and import the bundled schema.
  6. Request and import the unique .env and activation.json files.

Manual fallback paths

  • Windows launcher: ./Start-Installer-for-windows.bat
  • Linux launcher: ./Start-Installer-for-linux.sh
  • macOS launcher: ./Start-Installer-for-macos.command
  • Browser fallback: ./windows/gui/index.html, ./linux/gui/index.html, or ./macos/gui/index.html
