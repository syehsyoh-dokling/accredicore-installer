Why this is required

  1. The real installer must check Docker, PostgreSQL, Git, Node.js, and local write permissions.
  2. Browsers are blocked from running BAT, SH, or COMMAND files for security reasons.
  3. The native launcher opens the same installer screen, but with system access enabled.

Safe start steps

  1. Make sure the ZIP file has been extracted first.
  2. Open the extracted folder named Arab-Compaliance-Hub-Installer.
  3. Run the launcher file recommended above.
  4. When the installer opens, Runtime must show electron, not browser.
