## Recommended flow
1. Prepare or download your source ZIP.
2. Extract it into the intended project folder:
   - C:\Users\Saifuddin\accredicore-main
3. Review the extracted files.
4. Commit and push the updated project.

## Important note
Source code does not have to be placed inside XAMPP htdocs.
Use htdocs only if your runtime stack specifically requires Apache/PHP from that location.
Otherwise, the project can live in any folder you choose.
