AccrediCore Installer - Read First
=================================

Do not run index.html directly from inside the ZIP preview.

Correct Windows steps:
1. Open your Downloads folder.
2. Find accredicore-installer-final.zip.
3. Right-click the ZIP file.
4. Choose Extract All.
5. Open the extracted folder named Arab-Compaliance-Hub-Installer.
6. Double-click Start-Installer-for-windows.bat.
7. If Windows blocks the launcher, right-click Start-Installer-for-windows.bat and choose Run as administrator.

Correct macOS steps:
1. Extract the ZIP file.
2. Open Terminal inside the extracted folder.
3. Run:
   chmod +x ./Start-Installer-for-macos.command
   ./Start-Installer-for-macos.command

Correct Linux steps:
1. Extract the ZIP file.
2. Open Terminal inside the extracted folder.
3. Run:
   chmod +x ./Start-Installer-for-linux.sh
   ./Start-Installer-for-linux.sh

Browser fallback:
- index.html is only a guide and browser fallback.
- A normal browser cannot run native requirement checks.
- Use the native launcher for the full installation workflow.
