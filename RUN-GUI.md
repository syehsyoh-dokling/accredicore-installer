# Run AccrediCore GUI from root

This launcher keeps the GUI source in:

- windows/gui/index.html
- windows/gui/styles.css
- windows/gui/app.js

But lets you run it directly from the project root.

## Run from PowerShell

```powershell
powershell -ExecutionPolicy Bypass -File .\Run-AccrediCoreGui.ps1
```

## Direct path

```text
ROOT\Run-AccrediCoreGui.ps1
```
